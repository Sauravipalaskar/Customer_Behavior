DATA ANALYTICS PROJECT
Overview:
This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI.
It covers data loading, exploratory data analysis (EDA), data cleaning, querying with SQL, and
building an interactive dashboard. The final insights are summarized in a report and presentation.
Dataset:
Source: User-provided dataset
Format: CSV
Description: Structured dataset for analysis and visualization.
Tools & Technologies:
Python, Jupyter Notebook, SQL (PostgreSQL/MySQL/SQL Server), Power BI, Gamma, GitHub
Project Steps:
1. Data Loading
2. Exploratory Data Analysis
3. Data Cleaning
4. SQL Analysis
5. Dashboard Creation
6. Report & Presentation
Results:
Identified trends, patterns, and key insights for decision-making.
How to Run:
Clone repository, open notebook, install libraries, and run cells.
